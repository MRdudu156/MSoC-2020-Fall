## Usage
1. fpga board setup

We use **Xilinx ZedBoard Evaluation and Development Kit** to evaulate this project

2. run script.tcl
```sh
 vivado_hls script.tcl
```
3. Use vivado2019.2 build the system 
4. Run host program on jupyter notebook (your_zedboard_ip:9090)
